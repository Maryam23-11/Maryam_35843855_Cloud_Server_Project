Femme Wardrobe – E-Commerce Website

Femme Wardrobe is a fully functional e-commerce website built to showcase and sell women’s fashion products online. The project was developed using HTML, CSS, and JavaScript, and it is deployed on Amazon EC2 with a custom domain purchased from Namecheap. The website highlights both web development and cloud deployment skills, focusing on responsive design and secure hosting.

Project Overview

The website provides an engaging shopping experience, allowing users to browse, view, and interact with various fashion products. It features multiple pages, including a homepage, product listings, and a contact page, all designed to be visually appealing and user-friendly.

The deployment involved setting up an EC2 server on AWS, installing the Apache Web Server, and configuring SSL certificates via Command Prompt to ensure a secure HTTPS connection for the Namecheap domain.

Key Features:

Responsive layout optimized for desktop, tablet, and mobile users

Homepage with promotional banners and featured product sections

Product pages with images, detailed descriptions, and pricing

Navigation bar and footer for easy access across pages

Secure HTTPS access with SSL setup on Apache

Live hosting on AWS EC2 with domain mapping from Namecheap

Technologies Used:

HTML5 – structure and content of web pages

CSS3 – styling and responsive layouts

JavaScript – interactivity and dynamic elements

Apache Web Server – hosting and configuration

AWS EC2 – cloud deployment and server management

Namecheap – domain registration and DNS configuration

Let’s Encrypt SSL – free SSL installed via Command Prompt

GitHub – version control and project submission

Deployment Summary

Created and configured an Amazon EC2 instance running Amazon Linux

Installed Apache Web Server and transferred project files using SCP (Secure Copy Protocol)

Linked the Namecheap domain to the EC2 instance using public IP and DNS records

Generated and installed a free SSL certificate using Certbot through Command Prompt

Verified secure website access via HTTPS and tested all pages for functionality
